# Release 0.7.0: Image Editing

Status: Complete

## Goal

Edit an existing render through natural-language instructions while retaining
the source image as visual context and preserving durable version lineage.

## Included

- [x] Image-render parent/child lineage migration
- [x] Instruction-based Gemini image editing
- [x] Existing render supplied as visual context
- [x] Persistent edit instructions and output files
- [x] Repeated edits from any selected version
- [x] Side-by-side version comparison
- [x] Tests and optimized native verification

## Explicitly excluded

- Brush, lasso, or mask authoring
- Automatic prompt chat
- Bulk editing
- Timeline and export workflows

## Acceptance notes

- Editing never overwrites its source render.
- Every edit records its parent version and instruction.
- A user may select an older version and create a new branch from it.
- Render files remain inside controlled project folders.
