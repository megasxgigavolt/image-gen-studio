# Release 0.5.0: Image Workspace

Status: In progress

## Goal

Provide durable three-layer prompts, secure provider configuration, single-image
generation, and per-still version history.

## Included

- [x] Provider and legacy workflow audit
- [ ] Windows Credential Manager integration
- [ ] Configurable OpenAI and Gemini models
- [ ] Three-layer image settings
- [ ] Prompt and render version history
- [ ] Single Gemini image generation
- [ ] Functional Prompt and Settings tabs
- [ ] Tests and native verification

## Explicitly excluded

- Concurrent bulk generation
- Image editing/inpainting
- Chat refinement
- Reference-image conditioning
