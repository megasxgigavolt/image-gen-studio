# Release 0.4.0: Visual Plan

Status: Complete

## Goal

Generate and persist sentence-timed visual groups from saved source material,
then allow chronological regrouping with reset to the original plan.

## Included

- [x] Legacy segmentation audit and extraction boundary
- [x] Python engine contract for transcription and segmentation
- [x] SQLite migration for sentences, groups, and original-plan state
- [x] CPU-friendly local Whisper adapter with actionable dependency errors
- [x] Deterministic heuristic fallback grouping
- [x] Persistent sentence-level drag and drop
- [x] Automatic timestamp and duration derivation
- [x] Reset to the original generated plan
- [x] Generation progress and error states
- [x] Rust, Python, and frontend tests

## Explicitly excluded

- Manual word-level splitting
- AI visual metadata enrichment
- OpenAI scene-boundary analysis
- Excel as an internal transport
- Background job queue

## Acceptance criteria

- The engine contract returns typed word timestamps and scene groups.
- A plan can be generated without an OpenAI API call.
- Chronological order cannot be violated by drag and drop.
- Group timestamps derive from immutable sentence timestamps.
- Reset restores the exact originally generated grouping.
- Plan state survives application restarts.

## Timing note

Estimated timing is the default and is labeled in the interface. The engine
includes a CPU Whisper adapter behind an optional dependency; model installation
and bundled process orchestration remain separate deployment work.
