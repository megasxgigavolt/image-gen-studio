# Release 0.9.0: Timeline Foundation

Status: Complete

## Goal

Establish a durable interactive timeline model that assembles approved stills
against narration timing without pretending to be a full video editor.

## Included

- [x] Persistent timeline and clip SQLite migration
- [x] Build/reset timeline from the current visual plan
- [x] Still clips aligned to source sentence timing
- [x] Select and adjust clip boundaries within safe constraints
- [x] Zoom and playhead foundation
- [x] Disabled previews for captions, animation, transitions, and audio mixing
- [x] Tests and Windows installer verification

## Acceptance notes

- Timeline clips are deterministically derived from visual-plan sentence timing.
- Clip boundary updates reject invalid durations and overlaps.
- Playhead and zoom persist locally with safe bounds.
- Future editor controls are visible but disabled and perform no work.

## Explicitly excluded

- Video rendering or encoding
- Caption generation
- Animation generation
- Transitions and audio mixing
- Multi-track editing

## Invariants

- Clip source timing remains traceable to visual-plan groups.
- Clips may not overlap or have negative duration.
- Reset deterministically rebuilds the timeline from the visual plan.
- Disabled future controls perform no hidden work.
