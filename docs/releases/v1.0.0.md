# Release 1.0.0: Personal Stable

Status: Complete

## Goal

Deliver a hardened personal Windows release that safely opens existing projects,
survives recoverable local-database failures, and ships through verified
installer and portable artifacts.

## Included

- [x] Database integrity check and timestamped recovery backup
- [x] Startup diagnostics and user-readable recovery notice
- [x] Stable installer and portable executable artifacts
- [x] Release/version metadata audit
- [x] Full frontend, Rust, Python, build, and launch regression
- [x] Updated README, roadmap, changelog, and testing instructions

## Explicitly excluded

- Cloud sync, accounts, and teams
- Full video encoding/editor features
- Captions, thumbnails, metadata, and publishing
- Commercial licensing and automatic updates

## Stability invariants

- Recovery never silently deletes the original database.
- API keys remain in Windows Credential Manager.
- Exported projects contain no credentials.
- Existing v0.x data migrates forward automatically.
- The shipped installer launches a responsive window on Windows 10/11.
