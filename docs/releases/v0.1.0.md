# Release 0.1.0: Foundation

Status: Complete

## Goal

Create a maintainable application foundation that renders the agreed product
experience and can accept native and Python capabilities without restructuring.

## Included

- [x] Architecture decision and release roadmap
- [x] Monorepo directory structure
- [x] React and TypeScript application shell
- [x] Light and dark design tokens
- [x] Home, Inputs, Visual Plan, and Images navigation
- [x] Visual plan drag-and-drop prototype migrated into React
- [x] Typed mock domain data
- [x] Error boundary and application logging interface
- [x] Unit tests for visual-plan timestamp derivation
- [x] Lint, type-check, test, and production build scripts
- [x] Tauri shell configuration
- [x] GitHub Actions quality workflow

## Explicitly excluded

- SQLite persistence
- Provider API calls
- Python sidecar
- Real transcription
- Real image generation
- Portable executable
- Automatic updates

## Acceptance criteria

- The frontend starts with one command.
- All four current mockup stages are navigable.
- Sentence regrouping enforces chronological order and updates timestamps.
- Light and dark themes are usable.
- Type checking, unit tests, and production build pass.
- No API key or credential is stored in the repository.

- Vite production build passed
- Tauri optimized Windows build passed
- Native executable launch smoke test passed
- npm dependency audit reported zero vulnerabilities
