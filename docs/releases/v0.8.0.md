# Release 0.8.0: Export and Portability

Status: In progress

## Goal

Make finished stills and complete local projects portable, inspectable, and
recoverable without exposing provider secrets.

## Included

- [ ] Export selected image versions to a user-selected folder
- [ ] Export all latest stills with deterministic names and manifest
- [ ] Create a portable project bundle with database metadata and assets
- [ ] Import and validate a project bundle without overwriting existing data
- [ ] Bundle integrity checks and clear recovery errors
- [ ] Windows installer verification
- [ ] Tests and optimized native verification

## Explicitly excluded

- Cloud sync or accounts
- Provider credentials in exports
- Video timeline rendering
- Publishing and channel integrations

## Invariants

- Exports never include API keys.
- Imports receive fresh local identifiers when collisions exist.
- Existing projects are never overwritten silently.
- Partial import failures leave no half-created project.
